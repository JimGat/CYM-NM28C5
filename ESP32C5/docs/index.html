Release info

Loading manifest.

  • ROM mode: hold BOOT, plug the board in via USB-C, then release BOOT.
  • Close serial hogs: quit any terminal monitors or apps holding the COM port before connecting.
  • Stuck? Unplug, click Connect, then plug in again while holding BOOT.
  • After flash: press RESET if the board does not auto-reboot. Insert a FAT32 MicroSD card (≤ 32 GB) before booting.
  • Power: use a clean 5 V USB supply rated ≥ 500 mA. Undervoltage is the most common cause of reset loops during WiFi scanning.
